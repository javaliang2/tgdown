# 🌐 Web 界面快速版安装指南\n\n## ✨ 功能概览\n\n```\n✅ HTML5 视频播放器\n✅ 文件列表浏览\n✅ 分类筛选\n✅ 搜索功能\n✅ JWT 用户认证\n✅ 统计信息显示\n✅ 响应式设计（手机/电脑）\n```\n\n---\n\n## 🚀 快速开始（3 步）\n\n### 1️⃣ 安装依赖\n\n```bash\npip install Flask==2.3.3 Flask-CORS==4.0.0 PyJWT==2.8.0\n\n# 或更新整个依赖\npip install -r requirements-enhanced.txt\n```\n\n### 2️⃣ 配置 .env\n\n```bash\n# 添加以下配置到 .env 文件\n\n# Web 界面密码（使用明文）\nWEB_PASSWORD=admin\n\n# Web 界面端口\nWEB_PORT=5000\n\n# JWT 密钥（改为复杂字符串）\nJWT_SECRET=your-super-secret-key-change-me-in-production\n```\n\n### 3️⃣ 启动 Web 服务\n\n```bash\n# 方式 A：单独启动 Web 服务\npython3 web_api.py\n\n# 方式 B：使用 Gunicorn（生产推荐）\ngunicorn -w 4 -b 0.0.0.0:5000 web_api.py\n\n# 方式 C：Docker 启动\ndocker-compose -f docker-compose-web.yml up -d\n```\n\n### 4️⃣ 访问\n\n```\n浏览器访问：http://localhost:5000\n或：http://你的IP:5000\n\n首次登录密码：admin（来自 .env 中的 WEB_PASSWORD）\n```\n\n---\n\n## 🏗️ 项目结构\n\n```\nweb/\n├── index.html      # 主页面（HTML5）\n├── style.css       # 样式表（深色主题）\n├── api.js         # API 调用模块\n├── player.js      # 播放器逻辑\n\nweb_api.py        # Flask 后端服务\n```\n\n---\n\n## 🔐 认证说明\n\n### 登录流程\n\n1. 用户在 Web 界面输入密码\n2. 后端验证密码（与 `WEB_PASSWORD` 对比）\n3. 验证成功返回 JWT Token\n4. 前端存储 Token 到 `localStorage`\n5. 后续 API 请求自动携带 Token\n\n### 修改密码\n\n```bash\n# 在 .env 中修改\nWEB_PASSWORD=your-new-password\n\n# 重启 Web 服务\npython3 web_api.py\n```\n\n### 高级：使用密码哈希\n\n```bash\n# 生成密码哈希\npython3 -c \"import hashlib; print(hashlib.sha256(b'your-password').hexdigest())\"\n\n# 输出：abc123...\n\n# 在 .env 中配置\nWEB_PASSWORD_HASH=abc123...\n# （不需要设置 WEB_PASSWORD）\n```\n\n---\n\n## 📱 API 接口\n\n### 登录\n```bash\nPOST /api/login\n{\n  \"password\": \"admin\"\n}\n\n返回：\n{\n  \"success\": true,\n  \"token\": \"eyJ0eXAiOiJKV1QiLCJhbGc...\"\n}\n```\n\n### 获取文件列表\n```bash\nGET /api/files/{media_type}?token=YOUR_TOKEN&limit=100\n\n支持类型：photo, video, audio, document, animation\n\n返回：\n{\n  \"success\": true,\n  \"files\": [\n    {\n      \"id\": 1,\n      \"file_name\": \"video.mp4\",\n      \"file_size\": 1234567,\n      \"media_type\": \"video\",\n      \"downloaded_at\": \"2024-01-01 10:00:00\"\n    }\n  ]\n}\n```\n\n### 搜索文件\n```bash\nGET /api/search?q=keyword&token=YOUR_TOKEN&limit=50\n\n返回：\n{\n  \"success\": true,\n  \"files\": [...]\n}\n```\n\n### 获取统计信息\n```bash\nGET /api/stats?token=YOUR_TOKEN\n\n返回：\n{\n  \"success\": true,\n  \"stats\": {\n    \"total_files\": 100,\n    \"total_size\": 123456789,\n    \"by_type\": {\n      \"video\": {\"count\": 50, \"size\": 123456789},\n      \"photo\": {\"count\": 30, \"size\": 987654}\n    }\n  }\n}\n```\n\n### 获取下载链接\n```bash\nGET /api/download/{file_id}?token=YOUR_TOKEN\n\n返回：\n{\n  \"success\": true,\n  \"download_url\": \"/downloads/video.mp4\"\n}\n```\n\n---\n\n## 🌐 Nginx 反向代理\n\n### 配置示例\n\n```nginx\nserver {\n    listen 80;\n    server_name your-domain.com;\n\n    # Web 界面\n    location / {\n        proxy_pass http://localhost:5000;\n        proxy_set_header Host $host;\n        proxy_set_header X-Real-IP $remote_addr;\n    }\n\n    # 文件下载\n    location /downloads/ {\n        alias /path/to/downloads/;\n        autoindex on;\n    }\n}\n```\n\n---\n\n## 📊 页面功能\n\n### 登录页\n- 输入密码登录\n- 错误提示\n- 响应式设计\n\n### 主页\n- **左侧分类栏**：快速筛选文件类型\n- **顶部导航**：搜索框、统计、退出\n- **文件列表**：网格布局，点击播放\n- **播放器**：HTML5 视频播放，支持进度条、音量、全屏\n\n### 统计弹窗\n- 总文件数、总大小\n- 按类型统计\n- 实时更新\n\n---\n\n## 🎨 主题定制\n\n### 修改主题色\n\n编辑 `web/style.css`：\n\n```css\n:root {\n    --primary-color: #6366f1;  /* 主色 */\n    --bg-dark: #0f172a;        /* 背景 */\n    --text-primary: #e2e8f0;   /* 文本 */\n}\n```\n\n---\n\n## ⚡ 性能优化\n\n### 缓存\n- 使用 localStorage 存储 Token\n- 缓存用户搜索历史（可扩展）\n\n### 懒加载\n- 分页加载文件（每页 20 个）\n- 按需加载播放器\n\n### 压缩\n- 建议启用 Gzip 压缩\n- 使用 CDN 加速静态资源\n\n---\n\n## 🐛 故障排除\n\n### 无法登录\n```bash\n# 检查密码\necho $WEB_PASSWORD  #
